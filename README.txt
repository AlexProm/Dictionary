Ещё в процессе разработки - промежуточная версия... (Agile и всё такое)
Многое из того что хочется не реализованно (GUI, полноценная работа с одним файлом... Хотя может сделать app.py, из которого все блоки и запускать, стоит все функции вынести в отдельный файл/модуль...)

-> "Dictionary.py" - Непосредственно добавление, повторение слов
-> "DB.xls" - База слов в Документе Excel (Не исспользовал SQL базу данных, т.к. делал для себя и вообще лучше когда как можно больше пользователей будет уметь пользоваться компонентами приложения)
-> "back_Upper.py" - Копирование всего содержимого в соседнюю папку (с добавлением даты создания резервного хранилища)
-> "Перевод текста по базе слов.py" - Анализ текста, который находится в "TEXT.txt"  и сохраняет в "English Last Raport.txt"
-> "TEXT.txt" - Текст для Анализа и перевода (ну перевод будет когда наберётся достаточное количество слов в "DB.xls"
-> "English Last Raport.txt" - Запись последнего анализа текста
-> "Встречавшиеся неизвестные слова.txt" - Все известные неизвестные слова, которые бывали во всех текстах, проанализированных до этого
-> "Analytics.py" - Расчитывает и выводит на экран Возможные варианты развития базы слов
